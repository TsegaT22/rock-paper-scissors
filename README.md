# rock-paper-scissors
Web application that displays rock, paper, scissors game using React.
## To set this project up, put the following in the terminal:
1. `npm install`
2. `npm run dev`
3. Follow the local host link: http://localhost:5173/

## Reflection
I implemented client-side rendering using React + Vite effects including components, animations without any inner HTML coding. Everything happens in the server without using a backend.
